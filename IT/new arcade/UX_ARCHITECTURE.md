## UX ПРИНЦИПЫ

### 1. Прогрессивное раскрытие
```
Уровень 1: Главное меню → Быстрый доступ к основным функциям
Уровень 2: Профиль → Детальная информация о прогрессе
Уровень 3: Достижения → Подробности каждого бейджа
```

### 2. Визуальная обратная связь
```
Состояние             Визуальный эффект
─────────────────────────────────────────────
Normal                Зелёная рамка
Hover                 Яркое свечение + scale
Active                Инверсия цветов
Current User          Двойное свечение
Unlocked Achievement  Полная яркость + дата
Locked Achievement    50% прозрачность
```

### 3. Геймификационная мотивация
```
Элемент              Психологический эффект
──────────────────────────────────────────────────
Прогресс-бар         Близость цели → мотивация
Подиум               Социальное сравнение → конкуренция
Достижения           Коллекционирование → вовлечённость
Win Rate             Самоанализ → улучшение
Подсветка игрока     Личная значимость → эго
```

### 4. Информационная плотность
```
Экран              Плотность      Приоритет
──────────────────────────────────────────────
Главное меню       Низкая         Фокус и покой
Профиль            Средняя        Обзор прогресса
Лидерборд          Высокая        Максимум данных
Статистика игр     Высокая        Детальный анализ
```

## АДАПТИВНЫЕ БРЕЙКПОИНТЫ

```
Mobile (< 640px)
├── 1 колонка везде
├── Вертикальные кнопки
├── Компактные карточки
└── Уменьшенные шрифты (0.5-0.7rem)

Tablet (640px - 1024px)
├── 2 колонки в достижениях
├── Горизонтальные кнопки
├── Средние карточки
└── Базовые шрифты (0.6-0.8rem)

Desktop (> 1024px)
├── 3 колонки в профиле
├── 3 достижения в ряд
├── Полноразмерные карточки
└── Увеличенные шрифты (0.7-1rem)
```

## МИКРОИНТЕРАКЦИИ

```
Действие                     Реакция
────────────────────────────────────────────────────
Наведение на кнопку          Scale 1.05 + свечение
Клик по кнопке               Scale 0.95
Открытие профиля             Последовательное появление
Смена фильтра                Мгновенная пересортировка
Hover на достижении          Scale 1.05 + подсказка
Прокрутка лидерборда         Плавный скролл
Возврат назад                Fade out → fade in
```

## СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### A. Casual игрок (смотрит редко)
```
Путь: Главное меню → [Игра] → Главное меню
Иногда: → Профиль (проверить уровень)
```

### B. Целеустремлённый игрок
```
Путь: Главное меню → Профиль → Лидерборд → [Игра] → Профиль
Цикл: Проверка прогресса → Игра → Проверка прогресса
```

### C. Конкурентный игрок
```
Путь: Лидерборд → [Игра] → Лидерборд → [Игра]
Фокус: Постоянное отслеживание позиции в рейтинге
```

### D. Коллекционер достижений
```
Путь: Профиль (достижения) → [Игры] → Профиль
Цель: Открыть все бейджи
```
